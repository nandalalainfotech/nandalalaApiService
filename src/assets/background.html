How-to PDF Guides

How to edit PDF files

  1. Open a file in the Online PDF editor

  2. Click on the Text tool in the top menu

  3. Add text on the PDF page. Change text by clicking on existing text to start editing

  4. Add images to the page. Click and drag to move, resize or rotate the image

  5. Fill out PDF forms and add signatures. Draw, type or upload an image of your signature

  6. Annotate PDF pages, highlight text and mark changes with strikethrough

  7. Add new links to web URLs or pages in the document. Easily edit existing hyperlinks in the PDF

  8. Easily find and replace all occurrences of words in a PDF. Whiteout parts of the page. Add shapes

  9. Click Apply changes and download an edited document

Edit PDF files

How to convert a PDF to Word

  1. Open your document with the PDF to Word converter

  2. Click Convert and download a Word document

PDF to DOC converter

How to combine PDF files

  1. Open the Merge PDF files tool

  2. Drag to reorder the files, or sort them alphabetically

  3. Optionally select only a collection of pages to pick from each file

  4. Select your documents, or just drag and drop them all to the page

  5. Generate bookmarks or an outline based on the inputs

  6. Add file names to the footer of each page

  7. Generate a nice table of contents based on the documents being merged

  8. Optionally specify a cover / title page(s)

  9. Merge Acroforms or flatten form fields

Merge PDF files

How to convert PDF to JPG

  1. Open your PDF with the PDF to JPG, PNG and TIFF converter

  2. Convert all pages or select only pages you want converted

  3. Select desired image format, PNG, TIFF or JPG

  4. Choose a higher resolution, 150 or 220

  5. Quickly convert a PDF page to PNG or JPG

  6. Download an archive with all the converted images

PDF to image converter

How to delete pages from PDF

  1. Open the file with the Delete PDF Pages tool

  2. Click the pages you want to remove

  3. Download a new document, with the pages removed

Delete PDF Pages

How to edit PDF on Mac

  1. Edit PDF files online in your Safari browser

  2. Download and install Sejda Desktop and edit PDF documents offline

Edit PDF offline with Sejda Desktop for Mac

How to type on a PDF

  1. Open you file using the PDF editor

  2. Select the Text tool in the top menu

  3. Click anywhere on the page to start typing

  4. Fill out PDF forms easily, even if no form inputs are present

  5. Save and download your filled out document

Fill out PDF files

How to password protect a PDF

  1. Open the Encrypt and Protect PDF tool

  2. Select your PDF document

  3. Choose a really strong password (16 characters or more recommended)

  4. Optionally, select a set of restrictions for your document: modifying, printing, copying text and graphics, etc

  5. Save and download your protected PDF

Protect PDF with password and restrictions

How to compress a PDF

How to reduce PDF file size

  1. Open your document with the Compress PDF tool

  2. Configure image quality and resolution. This controls how small your PDF will become

  3. Compress and download your smaller PDF
